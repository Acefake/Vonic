# 文件管理插件

一个用于管理图片和文档文件的插件，支持导入、导出和分类管理。

## 功能

- **导入文件**：支持选择多个图片或文档文件，记录文件路径和元数据。
- **导出文件列表**：将文件列表导出为 JSON 备份。
- **文件分类**：自动识别图片（png、jpg、gif 等）和文档（pdf、docx、txt 等）。
- **文件管理**：查看文件详情、在文件夹中显示、删除文件。
- **扩展性**：支持随时添加新的文件格式。

## 使用方法

1. 安装插件后，在侧边栏中找到“文件管理”面板。
2. 点击“导入文件”按钮选择文件。
3. 文件将显示在列表中，并自动分类。
4. 可以点击“在文件夹中显示”打开文件所在目录。
5. 点击“导出列表”将当前文件列表保存为 JSON 文件。
6. 点击“清空”按钮删除所有记录。

## 开发

### 文件结构

```
file-manager/
├── manifest.json      # 插件清单
├── index.ts          # 插件主逻辑
├── Panel.vue         # 用户界面
├── package.json      # 依赖配置
└── README.md         # 说明文档
```

### 扩展支持的文件格式

编辑 `index.ts` 中的 `SUPPORTED_FORMATS` 对象，添加新的扩展名。

## API

插件提供以下 IPC 通道供其他插件或前端调用：

- `getAllFiles`：获取所有文件记录。
- `addFile`：添加一个文件记录。
- `removeFile`：删除一个文件记录。
- `clearFiles`：清空所有文件记录。
- `getSupportedFormats`：获取支持的文件格式。
