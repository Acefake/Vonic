# 待办清单

一个简单的待办事项管理插件。

## 功能

- 添加待办事项
- 标记完成/未完成
- 删除待办事项
- 数据本地持久化

## 使用方法

1. 启用插件后，在侧边栏可以看到待办清单面板
2. 在输入框中输入待办事项，点击添加按钮
3. 点击待办事项可以切换完成状态
4. 点击删除按钮可以移除待办事项

## 开发说明

本插件演示了 Vonic 插件系统的数据存储功能：

- `api.storage.get()` - 读取存储数据
- `api.storage.set()` - 保存存储数据
- `api.panels.register()` - 注册面板
