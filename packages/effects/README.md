## Effects 目录

`effects` 目录用于存放扩展插件和副作用相关的代码。

### 包列表

| 包名 | 说明 |
|------|------|
| `@vonic/plugins` | 扩展插件（ECharts 图表、Motion 动画） |

### 使用方式

```ts
// ECharts
import { EchartsUI, useEcharts } from '@vonic/plugins/echarts'

// Motion 动画
import { Motion, MotionPresets } from '@vonic/plugins/motion'
```
